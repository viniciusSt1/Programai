Instruções

O objetivo deste site é fornecer uma variedade de listas de exercícios correspondentes à disciplina de Laboratório de Programação I do curso de Engenharia da Computação da UFGD. Composto por cinco listas de exercícios, cada uma delas aborda tópicos específicos da ementa da disciplina. As listas contêm tanto exercícios teóricos (discursivos) quanto exercícios práticos (implementação de programas).

Aqui estão os tópicos abordados por cada lista de exercícios:

  1. Variável e seus tipos
  2. Operadores aritméticos
  3. Operadores relacionais
  4. Operadores lógicos
  5. Expressões lógicas
  6. Estruturas condicionais
  7. Laços de repetição
  8. Arrays
  9. Subrotina
  10. Ponteiros em C

A Lista 1 aborda os conteúdos dos tópicos 1, 2, 3 e 4, enquanto a Lista 2 aborda os conteúdos dos tópicos 5 e 6. Já as Listas 3, 4, 5 e 6 correspondem aos tópicos 7, 8, 9 e 10, respectivamente.

Cada lista de exercícios pressupõe conhecimentos adquiridos nas listas anteriores, portanto, é altamente recomendado seguir a sequência das listas caso você esteja aprendendo ou se familiarizando com os tópicos. Isso permitirá que você construa uma base sólida e desenvolva uma compreensão gradual dos conceitos abordados.

É importante mencionar que todas as listas de exercícios contêm tanto exercícios teóricos quanto práticos, e exigem conhecimentos fundamentais para todo programador.

  • Recomenda-se utilizar as dicas fornecidas apenas quando necessário.
  • A funcionalidade de ver a resposta é recomendada após tentar resolver o exercício ou para correção de uma resolução já feita.
  • Embora os exercícios tenham sido desenvolvidos com foco na linguagem C, é possível resolvê-los em qualquer outra linguagem de programação, o que pode ser benéfico para o seu aprendizado futuro.

Lista 1 - tópicos 1,2,3 e 4

  1. No contexto da programação, explique de forma simples e com suas próprias palavras o conceito de variáveis e o que podemos fazer com elas.

    As variáveis desempenham um papel fundamental na programação e são uma das primeiras coisas que aprendemos estudar programação. Explique o que ocorre quando criamos uma variável e discorra sobre as várias operações que podemos realizar com ela.

    As variáveis desempenham um papel fundamental na programação. Quando criamos uma variável, reservamos um espaço na memória do computador para armazenar um valor específico, que pode ser um número, uma palavra, um caractere ou outro tipo de dado. Essa capacidade de armazenamento nos permite realizar diversas operações entre variáveis, como realizar cálculos matemáticos, concatenar strings, comparar valores e muito mais. Além disso, as variáveis podem ser atualizadas e modificadas ao longo do programa, tornando nossos programas mais flexíveis e interativos.

  2. Liste os principais tipos de variáveis da linguagem C, juntamente com a quantidade de bytes que a máquina utiliza para armazená-las quando declaradas.

    Em linguagem C, existem quatro principais tipos de variáveis relacionados aos diferentes tipos de dados que podem ser armazenados. Cada tipo de variável possui um tamanho específico em bytes na memória do computador.

    Tipo "int" (inteiro): Utiliza 4 bytes de memória.

    Tipo "float" (ponto flutuante de precisão simples): Utiliza 4 bytes de memória.

    Tipo "double" (ponto flutuante de precisão dupla): Utiliza 8 bytes de memória.

    Tipo "char" (caractere): Utiliza 1 byte de memória.

  3. Em C, declare os principais tipos de variáveis e atribua um valor a cada uma delas. Em seguida, utilize o printf para exibir na tela o valor de cada variável

    Os principais tipos de variaveis são int, float, double e char

    No printf usamos:

    %d ou %i para mostrar numeros inteiros

    %f para tipo float

    %c para tipo char

    %if para tipo double

    
#include <stdio.h>

int main() {
    int numInteiro = 10;
    float numFloat = 3.14;
    double numDouble = 2.71828;
    char caractere = 'A';
                    
    printf("Valor do inteiro: %d\n", numInteiro);
    printf("Valor do ponto flutuante: %f\n", numFloat);
    printf("Valor do ponto flutuante de precisão dupla: %lf\n", numDouble);
    printf("Valor do caractere: %c\n", caractere);
                    
    return 0;
}

  4. Faça um programa em C que solicite ao usuário dois valores numéricos e armazene cada valor em uma variável diferente. Em seguida, troque os valores das variáveis, ou seja, atribua o valor da primeira variável à segunda e o valor da segunda variável à primeira. Por fim, utilize um único printf para exibir na tela as duas variáveis com seus valores trocados.

    Para realizar a troca de valores entre as variáveis utilize uma variável auxiliar. A variável auxiliar receberá o valor de uma das variáveis temporariamente, permitindo que não se perca o valor durante a troca.

    Use a função scanf(&variavel) para salvar os valores digitados pelo usuário

    
#include <stdio.h>

int main() {
    float valor1, valor2, temp;
        
    printf("Digite o primeiro valor: ");
    scanf("%f", &valor1);
        
    printf("Digite o segundo valor: ");
    scanf("%f", &valor2);
        
    // Troca dos valores
    temp = valor1;
    valor1 = valor2;
    valor2 = temp;
        
    printf("Valores trocados:\n");
    printf("Primeiro valor: %.2f\n Segundo valor: %.2f", valor1, valor2);
        
    return 0;
}

  6. Faça um programa em C que solicite ao usuário que informe dois valores numéricos. Em seguida, realize todas as operações aritméticas básicas entre esses valores e exiba o resultado de cada operação na tela.

    Para realizar as operações aritméticas (soma, subtração, multiplicação, divisão e módulo) entre os valores informados pelo usuário, utilize as variáveis adequadas e os operadores aritméticos correspondentes. Utilize a sintaxe correta para cada operação. Utilize o scanf para receber os valores do usuário. Certifique-se de armazenar cada valor em variáveis apropriadas, correspondentes ao tipo de dado que está sendo lido. Em seguida, utilize o printf para exibir cada resultado na saída do programa.

    
#include <stdio.h>

int main() {
    float valor1, valor2;
        
    // Solicita ao usuário que informe dois valores
    printf("Digite o primeiro valor: ");
    scanf("%f", &valor1);
        
    printf("Digite o segundo valor: ");
    scanf("%f", &valor2);
        
    // Realiza as operações aritméticas e exibe os resultados
    float soma = valor1 + valor2;
    float subtracao = valor1 - valor2;
    float multiplicacao = valor1 * valor2;
    float divisao = valor1 / valor2;
        
    printf("Soma: %.2f\n", soma);
    printf("Subtração: %.2f\n", subtracao);
    printf("Multiplicação: %.2f\n", multiplicacao);
    printf("Divisão: %.2f\n", divisao);
        
    return 0;
}

  7. Considere a equação matemática f(x) = 7.2x3-x + x/2 - 1. Escreva um programa em C que solicite ao usuário que informe o valor de x e, em seguida, calcule e exiba o resultado da função f(x) utilizando a fórmula fornecida.

    Ao calcular o valor da função f(x), é fundamental respeitar a ordem de precedência dos operadores matemáticos. Lembre-se de que as operações de exponenciação têm a maior prioridade, seguidas pelas operações de divisão e multiplicação, e por fim, as operações de adição e subtração. Utilize parênteses para agrupar as operações que devem ser realizadas primeiro, garantindo assim a correta avaliação das expressões.

    Utilize a bibliotecca math.h para fazer a exponenciação.

    
#include <stdio.h>
#include <math.h>
        
int main() {
    float x;
    float resultado;
        
    // Solicita ao usuário que informe o valor de x
    printf("Digite o valor de x: ");
    scanf("%f", &x);
        
    // Calcula o resultado da função f(x)
    resultado = pow(7.2, (3 - x)) + (x / 2) - 1;
        
    // Exibe o resultado da função f(x)
    printf("O resultado da função f(x) é: %.2f\n", resultado);
        
    return 0;
}

  8. Considere que uma caixa de bombom pode armazenar até 8 bombons. Escreva um programa em C que solicite ao usuário que informe a quantidade de bombons desejada. Em seguida, calcule e exiba a quantidade mínima de caixas necessárias para armazenar todos os bombons, levando em conta que cada caixa pode conter no máximo 8 bombons. Tente resolver o problema sem utilizar o comando 'if'.

    Utilize o conceito de arredondamento para cima (teto da divisão) para encontrar o número mínimo de caixas necessárias sem usar o comando 'if'. Divida a quantidade total de bombons pelo tamanho máximo de cada caixa (8) e armazene o resultado em uma variável inteira. Ao fazer isso, você obtém apenas a parte inteira do resultado, garantindo que qualquer quantidade parcial de bombons exija pelo menos uma caixa completa. Use a lógica para conseguir encontrar sempre o teto da divisão.

    
#include <stdio.h>

int main() {
    int quantidadeBombons;
    int quantidadeCaixas;
        
    // Solicita ao usuário que informe a quantidade de bombons desejada
    printf("Digite a quantidade de bombons desejada: ");
    scanf("%d", &quantidadeBombons);
        
    // Calcula a quantidade mínima de caixas necessárias
    quantidadeCaixas = (quantidadeBombons + 7) / 8;
        
    // Exibe a quantidade mínima de caixas necessárias
    printf("A quantidade mínima de caixas necessárias é: %d\n", quantidadeCaixas);
        
    return 0;
}

  10. Dado o trecho de código x = 1 'operador relacional' 0, qual será o valor de x para cada um dos operadores relacionais em C?

    Em outras linguagens de programação temos as variaveis do tipo lógico (false ou true). Em C essas variáveis são 0 e 1 respectivamente, ou seja, o valor de x será sempre 0 ou 1

    Igual a (==): Se o operador for ==, o resultado da expressão x = 1 == 0 será 0. Isso ocorre porque a igualdade 1 == 0 é falsa, e o valor atribuído a x será 0.

    Diferente de (!=): Se o operador for !=, o resultado da expressão x = 1 != 0 será 1. Isso ocorre porque a desigualdade 1 != 0 é verdadeira, e o valor atribuído a x será 1.

    Maior que (>): Se o operador for >, o resultado da expressão x = 1 > 0 será 1. Isso ocorre porque a comparação 1 > 0 é verdadeira, e o valor atribuído a x será 1.

    Maior ou igual a (>=): Se o operador for >=, o resultado da expressão x = 1 >= 0 será 1. Isso ocorre porque a comparação 1 >= 0 é verdadeira, e o valor atribuído a x será 1.

    Menor que (<): Se o operador for <, o resultado da expressão x = 1 < 0 será 0. Isso ocorre porque a comparação 1 < 0 é falsa, e o valor atribuído a x será 0.

    Menor ou igual a (<=): Se o operador for <=, o resultado da expressão x = 1 <= 0 será 0. Isso ocorre porque a comparação 1 <= 0 é falsa, e o valor atribuído a x será 0.

  11. Escreva quais são os operadores lógicos em C e sua respeciva sintaxe.

    Operadores lógicos são utilizados em programação para realizar operações de lógica booleana, ou seja, operações que envolvem valores lógicos verdadeiro (representado como 1) e falso (representado como 0). Esses operadores são utilizados para combinar ou inverter condições lógicas, permitindo a avaliação de múltiplas condições em uma expressão. Ao total temos 3 operadores lógicos em C

    E ou AND lógico

    Sintaxe: expressão1 && expressão2

    OU ou OR lógico

    Sintaxe: expressão1 || expressão2

    Não ou NOT lógico

    Sintaxe: !expressão

  12. Explique a diferença entre operadores aritméticos, operadores lógicos e operadores relacionais, e quando devemos utilizá-los.

    Para entender a diferença entre operadores aritméticos, operadores lógicos e operadores relacionais, leve em consideração o tipo de dados que estamos manipulando em cada caso.

    Os operadores aritméticos são utilizados para realizar operações matemáticas, como adição (+), subtração (-), multiplicação (*) e divisão (/). Eles são aplicados a valores numéricos e retornam um resultado calculado com base nos operandos.

    Os operadores relacionais são utilizados para comparar valores e verificar relações entre eles. Eles incluem operadores como maior que (>), menor que (<), maior ou igual a (>=), menor ou igual a (<=), igual a (==) e diferente de (!=). Esses operadores retornam um valor booleano (verdadeiro ou falso) com base na comparação realizada.

    Os operadores lógicos em linguagens de programação, como C, são usados para comparar valores booleanos (que podem ser gerado através dos operadores relacionais) e avaliar condições compostas. Eles conseguem associar expressões relacionais e lógicas gerando assim novos valores lógicos. Os operadores lógicos mais comuns são o "E" lógico (&&), o "OU" lógico (||) e o "NÃO" lógico (!).
